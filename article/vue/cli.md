# vue-cli
## 安装vue-cli脚手架

	npm install vue-cli -g

## 创建新项目

	vue init webpack my-project

## 安装依赖

	cd my-project
	npm install
	npm run dev

在浏览器输入`localhost:8080`查看

## 编译成浏览器可读的代码

	npm run build
